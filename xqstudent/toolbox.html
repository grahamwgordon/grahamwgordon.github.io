The commutative property of addition states that \(a+b=b+a\) for all real numbers \(a\) and \(b\).

You might accidentally switch the order of quantities being subtracted, but \(a-b\) is not necessarily the same as \(b-a\). You can avoid this by rewriting the difference as a sum: \(a-b = a+(-b)\). Then, if you want, you can apply the commutative property of addition the sum as follows: \(a+(-b) = (-b) + a\).

1. Start with an equation. \(4x + 3 = 5\)
2. Choose a quantity to add to each side. \(1\)
3. Add the quantity to both sides of the equation. \((4x + 3) + 1 = (5) + 1\)
4. The resulting equation has the same solution(s) as the original equation. \(4x+4=6\)

  • \(2x-1=3 \longrightarrow (2x-1)+1=(3)+1 \longrightarrow 2x = 4\)

The addition property states that adding the same quantity to both sides of an equation results in an equivalent equation. Recall that an equivalent equation is one which has the same solution(s) as the original. In symbols you can write this as \(a = b \implies a+c = b+c\) for all real numers \(a\), \(b\), and \(c\).
